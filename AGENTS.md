# Hungarian translation for game Foundation

The game is about managing a village/city during medieval ages and making villagers happy.

You can build a castle and a monastery for extra spice for the game

## Translations

Translations should have some medieval spice, but have to be clearly understandable for modern players. No accents, no local language allowed (speakings like Szeged, Székelyföld, etc are prohibited).

Translation of game-specific expressions, words (like mechanics, ranks, etc) should be recorded to @Translation-memory.md for keeping 
consistency in translation. Also, during translation, entries in @Translation-memory.md should take higher precedence for exactly same
reason.

## Format

The language file format is JSON, containing multiple objects with keys and values. Key translation is strictly prohibited, keys should be kept as-is as they are in the file.
